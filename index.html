餐廳等待區互動系統 prototype

壽司接接樂

一位牆前主玩家直接觸牆控制托盤接壽司，其他等待中的顧客以手機加入同一局，送出壽司與支援效果。

開啟大牆端 開啟手機端

專案結構

  • index.html入口頁
  • wall.html大牆端畫面
  • mobile.html手機端畫面
  • style.css共用樣式
  • wall.js大牆端遊戲邏輯
  • mobile.js手機端互動邏輯
  • README.md本地預覽說明

頁面清單

  • 大牆端：待機 / 進行中 / 回合結束 / 叫號提示
  • 手機端：加入首頁 / 已加入等待頁 / 支援操作頁 / 結果頁

本地預覽

  1. 解壓縮後進入資料夾。
  2. 直接雙擊 index.html 可先看靜態畫面。
  3. 要測試大牆端與手機端同步，建議用本地伺服器開啟。
  4. 同時開啟 wall.html 與 mobile.html 兩個頁面。
  5. 在手機端按下支援按鈕，即可看到大牆端同步變化。
